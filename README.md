# README

This README would normally document whatever steps are necessary to get the
application up and running.

Rails version: Rails 5.0.1
DataBase: PostgreSQL.

* La aplicación se llama definitivas, es una aplicación realizada en Ruby on rails 
  que tiene como objetivo falicitar a los profesores el calculo de las notas definitivas
  de los estudiantes.
* Para desplegar la aplicación usted debe descargar este repositorio, entrar a la carpeta
  descargada y ejecutar el siguiente comando $ bin/rails server .
* El comando crea un localhost en su computador, está conectado al puerto 3000 y ya puede 
  hacer uso de la aplicación, solo debe ingresar el nombre del estudiante, las 3 notas y su
  respectivo porcentaje. Finalmente le da click al botón de "Save data" y esté lo llevará 
  a una página en la que se listan las personas ingresadas a la base de datos.
